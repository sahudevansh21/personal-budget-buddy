# Personal Budget Buddy

**Live Demo:** [https://personal-budget-buddy-a6ajw4rmq-sahudevansh21-4649s-projects.vercel.app](https://personal-budget-buddy-a6ajw4rmq-sahudevansh21-4649s-projects.vercel.app)

## Problem
Many individuals struggle to visualize their daily or weekly spending habits and budget effectively, often leading to overspending or difficulty saving. Existing digital solutions typically require linking bank accounts or complex setups, which can deter users wary of sharing sensitive financial data.

## Solution
Personal Budget Buddy offers a straightforward, private, and secure way to manage your personal finances. This website allows users to manually input their income and expenses, categorize them, and view simple visualizations of their spending patterns over custom periods. All data is stored securely in the user's browser via local storage, enabling robust personal finance tracking without external APIs or privacy concerns.

## Features
-   **Dashboard**: Overview of your current balance, total income, total expenses, and recent transactions.
-   **Add Transaction**: Easily log new income or expense items with categories and descriptions.
-   **Budget Planner**: Set monthly budgets for different categories and track your spending against them.
-   **Transaction History**: View, filter, and manage all your past transactions.
-   **Secure & Private**: All your financial data is stored locally in your browser's storage, ensuring complete privacy.
-   **Stunning UI**: A dark-themed interface with vibrant gradient accents, glassmorphic cards, and smooth animations for an enjoyable user experience.
-   **Responsive Design**: Works seamlessly across various devices and screen sizes.

## Getting Started
To run the Personal Budget Buddy application locally, follow these steps:

1.  **Clone the repository (or extract these files into a new directory):**
    ```bash
    git clone <repository-url>
    cd personal-budget-buddy
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

    Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

4.  **Build for production:**
    ```bash
    npm run build
    ```

    This command optimizes your application for production.

5.  **Start the production server:**
    ```bash
    npm start
    ```

    This will serve the production-ready application.
